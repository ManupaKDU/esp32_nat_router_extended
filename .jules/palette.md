## 2026-05-17 - Appending missing CSS classes safely\n**Learning:** The project uses a purged/minified Bootstrap CSS file. If we want to use new Bootstrap classes like 'spinner-border' or 'visually-hidden' we can't just append them to the HTML; they need to be added to the compiled CSS. Trying to parse unminified CSS with regex can break things, so appending the minified representation of the classes directly to the end of the .css file is a safe workaround.\n**Action:** Manually append the exact minified CSS string to the end of the minified stylesheet when adding classes that were previously purged.
